# BGM Audio Files

Place MP3 files here for each floor's background music:

| File | Floor | Theme |
|------|-------|-------|
| `bgm1.mp3` | 1 - Ancient Ruins | mysterious, ambient |
| `bgm2.mp3` | 2 - Ice Cave | cold, ethereal |
| `bgm3.mp3` | 3 - Lava Corridor | dark, ominous |
| `bgm4.mp3` | 4 - Dark Forest | eerie, unsettling |
| `bgm5.mp3` | 5 - Abyss Temple | grand, deep |
| `bgm6.mp3` | 6 - Phoenix Furnace | intense, fiery |

## Recommended free sources (royalty-free)

- https://freesound.org/ (search: "dungeon ambient", "dark cave", etc.)
- https://incompetech.com/ (Kevin MacLeod, CC BY)
- https://www.zapsplat.com/ (free tier available)

If a file is missing, that floor plays in silence (no errors).
